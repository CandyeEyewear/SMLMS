## How Content Block Editing Works

### User Flow

1. User clicks **[+ Add Block]** button
2. User sees a **visual picker** with block type icons
3. User selects a block type (e.g., "Bullet List")
4. User sees a **friendly form** with labeled fields
5. User fills in the form using inputs, dropdowns, drag-drop
6. User clicks **[Save]**
7. System converts form data to JSON and stores in database
